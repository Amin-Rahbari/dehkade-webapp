# 📰 Dehkade News System
### سیستم مدیریت اخبار دهکده

---

## 📌 About

Dehkade News System is a secure news management web application developed using Flask and SQLite.

سیستم دهکده خبر یک وب‌اپلیکیشن مدیریت اخبار است که با Flask و SQLite طراحی شده است.

این پروژه به عنوان پروژه نهایی دوره کاردانی کامپیوتر نرم‌افزار طراحی شده است.

---

## ⚙️ Technologies

- Python
- Flask
- SQLite
- HTML/CSS
- Tailwind

---

## 🔐 Security Features

- Password Hashing
- CSRF Protection
- Secure File Upload
- Role Based Access Control
- Session Security
